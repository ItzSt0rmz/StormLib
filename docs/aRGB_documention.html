aRGB Strand Documentation

Constructor

stormlib::aRGB::aRGB(int adiPort, int length)

Constructs a new aRGB strand with the specified ADI port and length.

Parameters:

  • adiPort: The ADI port used for the LED strand.
  • length: The length of the LED strand.

Methods

void off()

Turns off the LED strand.

void flow(u_int32_t color1, u_int32_t color2, int speed)

Sets the LED strand to flow from one color to another.

Parameters:

  • color1: The starting color of the flow.
  • color2: The ending color of the flow.
  • speed: The speed of the color transition.

void flash(u_int32_t color, int speed, u_int32_t color2)

Sets the LED strand to flash a specific color.

Parameters:

  • color: The color to flash.
  • speed: The speed of the flash.
  • color2: The secondary color to flash in an alternating pattern.

void breathe(u_int32_t color)

Sets the LED strand to "breathe" a color, a slower, less harsh flash effect.

Parameters:

  • color: The color to breathe.

void init()

Initializes the main loop and updater tasks.
